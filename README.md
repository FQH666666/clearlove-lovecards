# clearlove-lovecards
## clearlove 清简表白墙，一款现代化，轻量化，便捷化的表白墙网站源码项目，基于php开发。

#### 演示站 http://test1.fangqihang.cn/

## 核心功能
#### 1. 双数据库兼容

    支持SQLite和MySQL两种数据库选择
    安装时提供数据库类型选择，适合不同环境需求
    自动适配数据库语法，确保跨数据库兼容性

#### 2. 主题系统

    25种常规主题配色，涵盖各种风格
    10种节日主题，带有独特的背景效果和动画
    支持自定义主题颜色，通过调色板选择
    主题自动切换功能：每天切换、随机切换
    节日自动识别并切换对应主题

#### 3. 管理功能

    完整的后台管理界面，支持帖子、评论管理
    公告和话题管理，增强社区互动
    敏感词审查系统，支持纯净模式
    数据统计与可视化，包括访问量和存储使用情况
    移动端适配，支持侧边栏自动收纳

#### 4. 用户体验

    响应式设计，适配各种设备屏幕
    流畅的动画效果，提升视觉体验
    智能搜索功能，快速定位表白内容
    媒体上传支持，丰富表达形式
    实时点赞和评论功能

#### 5. 技术特性

    使用PHP + PDO实现数据库操作
    集成LunarPHP库实现农历节日识别
    使用Tailwind CSS构建现代化界面
    Chart.js实现数据可视化
    模块化代码结构，易于维护和扩展
    自动将上传图片转换为WebP格式，减小文件大小，提升加载速度
    支持小带宽服务器的快速媒体加载
    响应式图片加载，根据设备自动调整图片质量
    懒加载技术，优化页面加载性能

#### 版本更新

    支持从旧版本升级，保留数据完整性
    提供自动升级脚本，简化升级过程

### 系统要求

    PHP 7.4+
    PDO扩展
    SQLite或MySQL数据库
    支持文件上传的服务器环境
