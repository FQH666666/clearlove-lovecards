# 校园表白墙系统

## 项目简介

这个校园表白墙系统是一个专为学校、社区等场所设计的在线互动平台，旨在为用户提供一个安全、便捷的情感表达空间。系统支持双数据库兼容（SQLite和MySQL），具有丰富的主题选择、节日自动切换、关键词审查等功能，为用户带来良好的使用体验。

## 核心功能

### 1. 双数据库兼容

- 支持SQLite和MySQL两种数据库选择
- 安装时提供数据库类型选择，适合不同环境需求
- 自动适配数据库语法，确保跨数据库兼容性

### 2. 主题系统

- 25种常规主题配色，涵盖各种风格
- 10种节日主题，带有独特的背景效果和动画
- 支持自定义主题颜色，通过调色板选择
- 主题自动切换功能：每天切换、随机切换
- 节日自动识别并切换对应主题

### 3. 管理功能

- 完整的后台管理界面，支持帖子、评论管理
- 公告和话题管理，增强社区互动
- 敏感词审查系统，支持纯净模式
- 数据统计与可视化，包括访问量和存储使用情况
- 移动端适配，支持侧边栏自动收纳

### 4. 用户体验

- 响应式设计，适配各种设备屏幕
- 流畅的动画效果，提升视觉体验
- 智能搜索功能，快速定位表白内容
- 媒体上传支持，丰富表达形式
- 实时点赞和评论功能

### 5. 技术特性

- 使用PHP + PDO实现数据库操作
- 集成LunarPHP库实现农历节日识别
- 使用Tailwind CSS构建现代化界面
- Chart.js实现数据可视化
- 模块化代码结构，易于维护和扩展
- 自动将上传图片转换为WebP格式，减小文件大小，提升加载速度
- 支持小带宽服务器的快速媒体加载
- 响应式图片加载，根据设备自动调整图片质量
- 懒加载技术，优化页面加载性能

## 应用场景

- **校园环境**：学生表达情感、交流互动的平台
- **企业内部**：员工交流、文化建设的工具
- **社区论坛**：邻里之间的情感连接
- **活动现场**：临时活动的互动墙

## 安装说明

1. 将项目文件上传到服务器
2. 访问 `install.php` 开始安装
3. 选择数据库类型（SQLite或MySQL）
4. 填写相应的数据库配置
5. 完成安装后，系统会自动跳转到首页

## 后台管理

- 访问 `admin.php` 进入后台登录
- 默认管理员账号：admin
- 默认管理员密码：admin123
- 登录后可进行各项配置和管理操作

## 系统要求

- PHP 7.0+
- PDO扩展
- SQLite或MySQL数据库
- 支持文件上传的服务器环境

## 安全建议

- 安装完成后删除 `install.php` 文件
- 定期备份数据库
- 及时更新敏感词库
- 为管理员账号设置强密码

## 版本更新

- 支持从旧版本升级，保留数据完整性
- 提供自动升级脚本，简化升级过程

这个系统不仅是一个表白墙，更是一个连接人与人情感的桥梁，通过现代化的技术和友好的界面，为用户创造一个温馨、安全的表达空间。
